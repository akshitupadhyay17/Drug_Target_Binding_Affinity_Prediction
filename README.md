# Drug Target Binding Affinity Prediction

This implementation is based on the research paper [**DeepDTA: Deep Drug–Target Binding Affinity Prediction**](https://doi.org/10.1093/bioinformatics/bty593) by Hakime Öztürk, Arzucan Özgür, and Elif Ozkirimli. It includes minor modifications to the model architecture and is implemented using TensorFlow.
